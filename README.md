**RASA CHATBOT**

_Rasa is an open source machine learning framework for building contextual AI assistants and chatbots. In this project, a simple Covid 19 FAQ Chatbot is built using RASA._

**What is included in this project?**

**1. FAQ about Covid**

-Only four questions are included for the sample.
- Using nlu.yml, stories.yml and domain.yml
- Using inbuilt features.

**2. Custom Action**

-  A custom action of getting a number of cases of covid is implemented.
- While doing this project, I couldn't find any updating API for regular updating.
- Used https://corona.askbhunte.com/api/v1/data/nepal link to show that real time data    can also be updated through this chatbot as a sample
All the installation requirement is in requirements.txt
